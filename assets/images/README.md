# Carpeta de Imágenes

## Logo de CashSpace

Por favor, sube tu logo aquí con el nombre: `logo.png` o `logo.svg`

### Requisitos:
- **Nombre del archivo:** `logo.png` o `logo.svg`
- **Formato recomendado:** PNG con fondo transparente
- **Tamaño recomendado:** 
  - Para navbar: 150-200px de ancho
  - Para login/register: 200-300px de ancho

### Nota:
Si el logo tiene fondo gris, puedes usar herramientas online como:
- https://www.remove.bg/ (para quitar fondos)
- https://photopea.com/ (editor online similar a Photoshop)

O si prefieres, puedo ayudarte a procesarlo una vez que lo subas.
